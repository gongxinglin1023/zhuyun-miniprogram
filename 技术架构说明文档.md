## 🔧 后端修改入口

### 1. 修改品牌配置

只需修改 `brand-config.json` 文件，即可全局更新：

| 配置项 | 说明 | 修改后生效位置 |
|--------|------|----------------|
| `brand.name` | 品牌名称 | 官网、小程序、公众号 |
| `brand.slogan` | 品牌口号 | 官网、小程序 |
| `brand.logo` | Logo地址 | 官网、小程序 |
| `contact.phone` | 联系电话 | 官网、小程序 |
| `contact.email` | 邮箱地址 | 官网 |
| `theme.primary` | 主色调 | 官网、小程序 |
| `theme.secondary` | 辅助色 | 官网、小程序 |

### 2. API接口切换

修改 `api-config.json`：

```json
{
  "api": {
    "baseUrl": "https://api.zhuyun.com"  // 改为实际后端地址
  },
  "mock": {
    "enabled": false  // 设为false切换到远程API
  }
}
```

### 3. 小程序配置

修改 `miniprogram/api/config.js`：

```javascript
export default {
  mock: {
    enabled: false  // 设为false使用远程API
  },
  api: {
    baseUrl: 'https://api.zhuyun.com'  // 实际后端地址
  }
}
```
